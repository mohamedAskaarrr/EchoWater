## 🚀 Quick Start

```bash
# Clone & Install
git clone https://github.com/mohamedAskaarrr/EchoWater.git
cd EchoWater/EchoWater
composer install && npm install

# Setup Environment
cp .env.example .env && php artisan key:generate
php artisan migrate

# Start Development
npm run dev
php artisan serve
```

Visit `http://localhost:8000` 🌐
